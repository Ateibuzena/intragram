# Intragram Frontend

Red social para estudiantes de 42 Málaga - Frontend con React + Vite + Tailwind CSS

## 🚀 Setup

```bash
# Instalar dependencias
npm install

# Copiar variables de entorno
cp .env.example .env

# Editar .env con tus credenciales de 42
nano .env

# Iniciar servidor de desarrollo
npm run dev
